## Objetivo do Projeto

Este repositório faz parte do desafio do curso **“Simulando um Malware de Captura de Dados Simples em Python e Aprendendo a se Proteger”**.

A proposta é:

- Simular o comportamento básico de um **ransomware** em arquivos de teste;
- Simular um **keylogger** simples em modo controlado;
- Refletir sobre **estratégias de defesa** e prevenção contra esse tipo de ameaça;
